What we need?
• A Heroku user account

Preparing the app #1

change in Gemfile.rb 
gem 'sqlite3'
to 
group :development do
  gem 'sqlite3'
end
group :production do
  gem 'pg'
end
run in cmd 
bundle install --without production
...
Preparing the app #2

add project to GIT 
git init

git add .

git commit -m "first commit"
...
Preparing the app #3

heroku gem 
gem install heroku


add to application.rb 
config.assets.initialize_on_precompile = false
...
Deploying #1

login 
heroku login
add ssh-key 
heroku keys:add c:\sites\.ssh\id_rsa.pub
            #or
heroku keys:add ~/.ssh/id_rsa.pub
create new app 
heroku create rails-girls-rocks
...
Deploying #2

push the code 
git push heroku master
migrate database 
heroku run rake db:migrate
run the app 
heroku open
http://rails-girls-rocks.herokuapp.com
...
Re-Deploy 

git add .

git commit -m "some comment"

git push heroku master


heroku run rake db:migrate
